## Getting Started

### Prerequisites
Ensure the following are installed:
* `node`

### Installation
Clone the repository
    `git clone https://github.com/adamdlear/hackathon-pets`

Change directory to the project folder
    `cd hackathon-pets`

Install dependencies
    `npm install`
